# 技术设计文档 — 基于本地 Ollama AI 的智能客服系统

## 1. 项目背景与目标
- **背景**：利用本地部署的 Ollama AI 模型，构建安全、高效的企业级智能客服系统。
- **目标**：
  - 支持多种格式文档（文本、图片）学习与知识库构建。
  - 具备问题解析、答案计算与拼接能力。
  - 支持上下文对话管理，提升交互体验。
  - 提供简洁易用的前端界面。

## 2. 系统架构设计

```
用户 <--> 前端界面 <--> 后端服务 <--> Ollama AI模型
                      |            |
                      |            --> 文档处理模块
                      |            --> 问题解析模块
                      |            --> 答案生成模块
                      --> 上下文管理模块
```

### 2.1 组件说明
- **前端界面**：基于当前的前端架构，提供聊天窗口、文档上传、历史记录查看等功能。
- **后端服务**：Node.js/Python Flask，负责业务逻辑处理、调用 Ollama API。
- **文档处理模块**：
  - 支持文本提取（OCR处理图片中的文字）。
  - 将文档内容转为结构化的 FAQ 格式存储（问题-答案对）。
- **问题解析模块**：
  - 对用户问题进行意图识别和关键词提取。
- **答案生成模块**：
  - 根据解析结果调用 Ollama AI 模型，融合知识库内容生成答案。
- **上下文管理模块**：
  - 维护对话上下文，支持多轮问答。

## 3. 详细设计

### 3.1 文档学习与知识库构建
- **输入**：多格式文档（PDF、Word、图片等）。
- **流程**：
  1. 文档上传。
  2. 图片通过 OCR 转文本。
  3. 文本内容抽取 FAQ 格式（如正则规则或模型辅助抽取）。
  4. 存入本地数据库（如 MYSQL）。
- **存储结构示例**：

| FAQ_ID | Question                | Answer                  | Source Document | Metadata       |
|--------|-------------------------|-------------------------|-----------------|----------------|
| 1      | 如何重置密码？          | 点击“忘记密码”链接...    | user_manual.pdf | 上传日期、作者 |

### 3.2 用户问题处理流程
- **步骤**：
  1. 用户输入问题。
  2. 问题解析模块进行意图识别和关键词抽取。
  3. 检索知识库中相关 FAQ。
  4. 调用 Ollama AI，结合检索结果和上下文，生成答案。
  5. 返回答案给用户。

### 3.3 上下文管理
- 维护会话 ID，存储最近 N 轮对话。
- 在调用 AI 生成答案时，将上下文作为输入，提升回答连贯性。

### 3.4 前端设计
- 聊天窗口展示问答。
- 文档上传界面。
- 会话历史查看。
- 状态提示（加载、错误等）。

## 4. 技术选型

| 功能         | 技术方案                               |
|--------------|------------------------------------|
| 前端         | 当前这个框架结构                           |
| 后端         | Node.js (Express) / Python (Flask) |
| 数据库       | MYSQL                              |
| OCR          | Tesseract OCR                      |
| AI模型调用   | 本地 Ollama API                      |
| 文档解析     | 自定义规则 + NLP工具（如 spaCy）             |

## 5. 优化与改进建议

| 方面           | 建议说明                                         |
|----------------|------------------------------------------------|
| 文档学习       | 使用半自动化标注工具，提升 FAQ 抽取质量。       |
| 多模态支持     | 除文本OCR，考虑图像理解模型辅助理解图片内容。   |
| 上下文管理     | 增加会话状态管理，支持用户纠错和上下文切换。     |
| 性能优化       | 缓存常见问题答案，减少 AI 模型调用次数。         |
| 用户体验       | 增加多轮引导式问答，提升用户交互流畅度。         |
| 安全性         | 本地部署确保数据隐私，增加访问权限控制。         |
