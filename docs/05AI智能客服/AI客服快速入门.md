# 🚀 智能客服系统 - 快速入门

## 5 分钟快速体验

### 第一步：安装依赖（1 分钟）

```bash
cd /Users/linziwang/PycharmProjects/wordToWord
pip install -r requirements-chatbot.txt
```

### 第二步：启动 Ollama（1 分钟）

**新终端窗口 1：**
```bash
ollama serve
```

**另一个新终端窗口 2：**
```bash
# 如果还没安装模型，拉取推荐模型
#ollama pull qwen3:8b
ollama pull deepseek-r1:7b
```

### 第三步：启动应用（1 分钟）

**回到项目主终端：**
```bash
cd /Users/linziwang/PycharmProjects/wordToWord
python app.py
```

或使用快速启动脚本：
```bash
./start_chatbot.sh
```

### 第四步：访问系统（1 分钟）

浏览器打开：
```
http://localhost:5001/chatbot
```

或从首页点击"智能客服"卡片。

### 第五步：开始使用（1 分钟）

#### 方式一：直接提问
```
在输入框输入：你好，介绍一下你自己
```

#### 方式二：上传文档学习
1. 点击左侧"文档库"标签
2. 拖拽一个文档到上传区
3. 等待自动提取 FAQ
4. 基于文档内容提问

#### 方式三：查看已有知识
1. 点击"FAQ"标签
2. 浏览已学习的问答对
3. 点击任意问题快速提问

## 提示与技巧

### 💡 提高回答质量

1. **明确问题**：问题越具体，回答越准确
   - ❌ "产品怎么样？"
   - ✅ "产品 A 的主要功能有哪些？"

2. **提供上下文**：多轮对话时保持连贯
   ```
   用户：公司产品线有哪些？
   AI: [列出产品]
   用户：第二个产品的价格？  # AI 知道"第二个产品"指代什么
   ```

3. **善用搜索**：先搜索 FAQ，再提问

### ⚡ 性能优化

- **缓存热门问题**：相同问题会优先匹配历史答案
- **批量上传**：一次性上传多个相关文档
- **选择合适模型**：qwen3:8b 适合中文场景

### 🔍 故障排查

**AI 不回答？**
1. 检查 Ollama 服务：`curl http://localhost:11434/api/tags`
2. 检查模型是否已安装：`ollama list`
3. 查看应用日志

**文档上传失败？**
1. 检查文件格式是否支持
2. 确认文件大小（建议 < 10MB）
3. 查看错误提示
