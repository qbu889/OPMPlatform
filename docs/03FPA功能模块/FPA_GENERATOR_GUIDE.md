# FPA预估表生成器使用说明

## 📋 功能概述

FPA预估表生成器是一个用于从需求规格说明书中自动提取功能点信息，并生成符合FPA(功能点分析)规范的Excel预估表的工具。

## 🎯 主要功能

- ✅ 自动识别需求文档的层级结构
- ✅ 提取功能描述、输入、输出等关键信息
- ✅ 统计内部/外部逻辑文件数量
- ✅ 生成标准化的 FPA预估 Excel 表格
- ✅ 支持拖拽上传和批量处理
- ✅ 自动生成汇总统计信息

## 📖 使用步骤

### 1. 访问 FPA预估表生成页面

在系统首页点击 **"FPA预估表生成"** 卡片，或直接访问 `/fpa-generator/` 页面。

### 2. 准备需求文档

确保你的需求文档符合以下格式要求:

#### 文档格式要求

```markdown
## 一级分类（如：监控管理应用）
### 二级分类（如：故障监控应用）
#### 三级分类（如：集团事件对接）
##### 功能点名称（如：网络事件业务影响）
###### 功能点计数项（如：家宽业务 OLT 脱管场景业务采集）

**功能描述：**进行家宽业务 OLT 脱管场景业务采集
**系统界面：**无
**输入：**系统自动实时触发
**输出：**查询出满足条件的事件数据
**处理过程：**系统自动实时触发，进行数据采集和处理

**本事务功能预计涉及到 1 个内部逻辑文件，0 个外部逻辑文件**

本期新增/变更的内部逻辑文件：家宽业务影响指标清单表
本期涉及原有但没修改的内部逻辑文件：网络事件表
本期新增/变更的外部逻辑文件：无
本期涉及原有但没修改的外部逻辑文件：无
```

#### 关键字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **功能描述** | 描述功能的主要作用 | 进行 xxx 业务处理 |
| **系统界面** | 是否涉及用户界面 | 无 / 有 |
| **输入** | 功能的输入数据或触发条件 | 系统自动实时触发 |
| **输出** | 功能的输出结果 | 查询出满足条件的事件数据 |
| **处理过程** | 详细的处理步骤 | 1.接收数据;2.处理;3.输出结果 |
| **内部逻辑文件数** | 涉及的数据文件数量 | 1 个内部逻辑文件，0 个外部逻辑文件 |
| **新增/变更内部逻辑文件** | 本期新增或修改的文件 | 家宽业务影响指标清单表 |
| **原有未修改内部逻辑文件** | 本期使用但未修改的文件 | 网络事件表 |

### 3. 上传文档

1. **点击上传**: 点击上传区域选择文件
2. **拖拽上传**: 直接拖拽.md 文件到上传框
3. 支持的文件格式：`.md`(Markdown)

### 4. 生成 FPA预估表

点击 **"开始生成 FPA预估表"** 按钮，系统将:
- 解析 Markdown 文档
- 提取所有功能点信息
- 自动统计文件数量
- 生成标准化 Excel 表格

### 5. 下载结果

生成成功后:
- 查看功能点统计信息
- 点击 **"下载 FPA预估表"** 按钮
- 保存 Excel 文件到本地

## 🔍 常见问题

### Q1: 文档解析失败怎么办？

**A:** 请检查以下几点:
1. 确保文档是 Markdown 格式 (.md 后缀)
2. 确保文档包含完整的层级结构 (##, ###, ####, #####, ######)
3. 确保功能描述字段格式正确 (**功能描述:** 后跟内容)
4. 检查文档编码是否为 UTF-8

### Q2: 为什么有些功能点没有解析出来？

**A:** 可能的原因:
1. 标题级别不正确 (应该使用 5-6 级标题)
2. 功能描述字段缺少或格式不对
3. 文档中存在特殊字符导致解析失败

### Q3: 如何修改生成的 Excel 表格？

**A:** 下载的 Excel 文件可以用 Microsoft Excel、WPS Office 或 LibreOffice Calc 打开并进行编辑。

### Q4: 支持批量处理吗？

**A:** 当前版本支持单个文档上传，如需批量处理多个文档，可以依次上传生成。

## 🛠️ 技术细节

### 解析规则

- **标题识别**: 
  - `##` → 一级分类
  - `###` → 二级分类
  - `####` → 三级分类
  - `#####` → 功能点名称
  - `######` → 功能点计数项

- **字段提取**: 使用正则表达式匹配加粗字段标签
- **文件统计**: 自动计算逗号、分号分隔的文件数量

### 数据验证

系统会自动验证:
- 必填字段是否存在
- 文件格式是否正确
- 文件路径是否合法
