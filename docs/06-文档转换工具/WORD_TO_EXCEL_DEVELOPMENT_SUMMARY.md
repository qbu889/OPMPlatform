**核心算法**:
```python
def _parse_hierarchy_number(self, text: str):
    """解析层级编号"""
    pattern = r'^(\d+(?:\.\d+)*)[\s、](.+)$'
    # 3.1.1.2 → 4 级 → 一级分类
    # 3.1.1.2.1 → 5 级 → 二级分类
    # 3.1.1.2.1.1 → 6 级 → 功能模块
    # 3.1.1.2.1.1.1 → 7 级 → 功能点名称
```

```

    ↓

    ↓
保存到临时目录
    ↓
WordDocumentParser.parse()
    ↓
遍历段落 → 识别编号 → 更新层级
    ↓
提取功能点 → 清洗描述
    ↓
生成 DataFrame (pandas)
    ↓
保存为 Excel (.xlsx)
    ↓
返回下载链接
    ↓
用户下载
```
