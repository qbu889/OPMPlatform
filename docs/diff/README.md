# JSON 对比工具使用说明

## 📖 功能介绍

JSON 对比工具是一个智能的 JSON 数据差异分析工具，能够：

- ✅ **忽略字段顺序**：自动识别相同字段，不受排列顺序影响
- ✅ **智能高亮**：绿色=相同，红色=不同，蓝色=新增/删除
- ✅ **实时对比**：支持自动对比和手动对比两种模式
- ✅ **文件上传**：支持直接上传 JSON 文件进行对比
- ✅ **统计信息**：显示相同、不同、新增、删除字段数量

## 🚀 使用方法

### 1. 访问页面

在导航栏中点击：**高效工具 → JSON 对比工具**

或直接访问：`http://localhost:5200/json-diff`

### 2. 输入数据

有两种方式输入 JSON 数据：

#### 方式一：粘贴文本
- 在左侧输入框粘贴第一个 JSON 数据
- 在右侧输入框粘贴第二个 JSON 数据

#### 方式二：上传文件
- 点击"上传文件"按钮选择 JSON 文件
- 支持 `.json` 格式的文件

### 3. 执行对比

- **手动模式**：填写完数据后点击"开始对比"按钮
- **实时模式**：开启"实时对比"开关，输入后自动对比（延迟1秒）

### 4. 查看结果

对比完成后会显示：

1. **统计信息**：总字段数、相同/不同/新增/删除数量
2. **左右对比视图**：分栏显示两侧数据的差异
3. **颜色标记**：
   - 🟢 绿色背景：字段值相同
   - 🔴 红色背景：字段值不同
   - 🔵 蓝色背景：仅存在于一侧（新增/删除）

## 💡 示例演示

点击"加载示例"按钮可以快速体验功能：

```json
// 左侧数据
{
  "name": "张三",
  "age": 25,
  "city": "北京",
  "email": "zhangsan@example.com"
}

// 右侧数据（字段顺序不同）
{
  "city": "北京",
  "name": "张三",
  "age": 26,           // 年龄不同（红色）
  "phone": "138..."    // 新增字段（蓝色）
}
```

对比结果：
- `name` 和 `city` 标记为绿色（相同）
- `age` 标记为红色（不同：25 vs 26）
- `email` 在左侧标记为灰色（已删除）
- `phone` 在右侧标记为蓝色（新增）

## 🎨 设计规范

本工具遵循 Apple 设计规范：

- **字体**：使用系统默认字体，负字间距
- **颜色**：
  - 相同：淡绿色背景 `#e1f3d8`
  - 不同：淡红色背景 `#fde2e2`
  - 新增：淡蓝色背景 `#e1f0ff`
  - 删除：浅灰色背景 `#f4f4f5`
- **圆角**：8px 标准圆角
- **阴影**：悬停时轻微提升效果

## 🔧 技术实现

### 后端
- **框架**：Flask Blueprint
- **路由**：`/api/diff/compare` (POST)
- **核心算法**：递归对比 JSON 树结构
- **位置**：
  - `routes/diff/diff_routes.py`
  - `utils/json_diff_utils.py`

### 前端
- **框架**：Vue 3 + Composition API
- **UI库**：Element Plus
- **组件**：`frontend/src/views/tools/JsonDiff.vue`
- **路由**：`/json-diff`

## 📊 性能指标

- 支持最大 5MB 的 JSON 文件
- 1000行以内对比响应时间 < 1秒
- 10000行以内对比响应时间 < 5秒

## ⚠️ 注意事项

1. **JSON 格式**：确保输入的 JSON 格式正确，否则会提示错误
2. **编码问题**：建议使用 UTF-8 编码的 JSON 文件
3. **嵌套深度**：支持任意深度的嵌套对象和数组
4. **数据类型**：支持字符串、数字、布尔值、null、对象、数组
