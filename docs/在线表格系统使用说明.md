# 📊 在线表格系统

一个类似飞书表格的在线协作电子表格系统，支持自定义列、数据编辑、样式设置等功能。

## ✨ 功能特性

### 🎯 核心功能
- ✅ 创建和管理多个表格
- ✅ 自定义列类型（文本、数字、日期、下拉选项）
- ✅ 实时编辑和保存
- ✅ 添加/删除行和列
- ✅ 单元格样式自定义
- ✅ 数据导出（CSV 格式）
- ✅ 操作历史记录
- ✅ 支持示例数据（BUG 跟踪表）

### 🎨 界面特点
- 🌈 现代化渐变 UI 设计
- 📱 响应式布局
- ⚡ 快速加载和渲染
- 🖱️ 直观的右键菜单操作
- 🎯 类似飞书表格的交互体验

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
# 1. 启动应用
./start_spreadsheet.sh

# 2. 访问系统
打开浏览器访问：http://localhost:5001/spreadsheet/
```

### 方法二：手动启动

```bash
# 1. 确保数据库已初始化（已初始化可跳过）
mysql -h localhost -P 3306 -u root -p12345678 fpa_rules < sql/init_spreadsheet.sql

# 2. 启动应用
python3 app.py

# 3. 访问系统
打开浏览器访问：http://localhost:5001/spreadsheet/
```

## 📖 使用指南

### 1️⃣ 创建表格

1. 访问表格列表页：`http://localhost:5001/spreadsheet/`
2. 点击"+ 创建新表格"按钮
3. 输入表格名称和描述
4. 系统会自动创建默认列（序号、项目名称、负责人、状态、优先级、备注）

### 2️⃣ 编辑表格

#### 添加行
- 点击表格底部的"+ 添加新行"按钮
- 或使用工具栏的"+ 添加行"按钮

#### 添加列
- 点击工具栏的"+ 添加列"按钮
- 输入列名和类型（text/number/date/select）

#### 编辑单元格
- 直接点击单元格进行编辑
- 下拉类型会自动显示选项供选择
- 数据会自动保存

#### 删除行/列
- 点击行尾的"删除"按钮删除行
- 鼠标悬停在列表头，点击出现的"×"按钮删除列

### 3️⃣ 导出数据

1. 打开要导出的表格
2. 点击工具栏的"导出"按钮
3. 数据将自动下载为 CSV 文件

## 🛠️ 技术栈

### 后端
- **框架**: Flask 2.x
- **ORM**: SQLAlchemy
- **数据库**: MySQL 8.0
- **语言**: Python 3.8+

### 前端
- **原生 JavaScript**: 无需框架，轻量级
- **CSS3**: 渐变、动画效果
- **HTML5**: 语义化标签

### 数据库表结构

```
spreadsheet          - 表格定义表
spreadsheet_column   - 列定义表
spreadsheet_row      - 行数据表
spreadsheet_cell     - 单元格数据表
spreadsheet_history  - 操作历史表
```

## 🔌 API 接口

### 表格管理
- `GET /spreadsheet/` - 表格列表页面
- `GET /spreadsheet/<id>` - 表格详情页面
- `POST /spreadsheet/api/create` - 创建表格
- `GET /spreadsheet/api/list` - 获取表格列表
- `GET /spreadsheet/api/<id>` - 获取表格信息
- `DELETE /spreadsheet/api/<id>` - 删除表格

### 列管理
- `GET /spreadsheet/api/<id>/columns` - 获取列定义
- `POST /spreadsheet/api/<id>/columns` - 添加列
- `PUT /spreadsheet/api/columns/<id>` - 更新列
- `DELETE /spreadsheet/api/columns/<id>` - 删除列

### 行管理
- `GET /spreadsheet/api/<id>/rows` - 获取行数据
- `POST /spreadsheet/api/<id>/rows` - 添加行
- `DELETE /spreadsheet/api/rows/<id>` - 删除行

### 单元格管理
- `GET /spreadsheet/api/cells/<id>` - 获取单元格
- `PUT /spreadsheet/api/cells/<id>` - 更新单元格
- `POST /spreadsheet/api/batch-update` - 批量更新

### 历史记录
- `GET /spreadsheet/api/<id>/history` - 获取操作历史
