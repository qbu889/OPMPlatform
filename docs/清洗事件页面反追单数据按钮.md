# 清洗事件页面添加反追单数据按钮

## 功能说明

在清洗事件页面的事件时间输入框中，新增了一个"反追单数据"按钮（显示为"-2天"），点击后会自动将当前时间减去两天并填充到时间输入框中。

## 实现细节

### 1. UI 修改

在事件时间输入框的 `append` 插槽中添加了新按钮：

```vue
<template #append>
  <el-button @click="insertCurrentTime" title="插入当前时间">
    <el-icon><Clock /></el-icon>
  </el-button>
  <el-button @click="subtractTwoDays" type="warning" title="反追单数据（减2天）">
    -2天
  </el-button>
</template>
```

**按钮特性**：
- 类型：`warning`（橙色），与"插入当前时间"按钮区分
- 提示文本：鼠标悬停时显示"反追单数据（减2天）"
- 显示文本："-2天"，简洁明了

### 2. 功能实现

新增 `subtractTwoDays` 函数：

```javascript
// 反追单数据：当前时间减两天
const subtractTwoDays = () => {
  const now = new Date()
  // 减去2天
  now.setDate(now.getDate() - 2)
  
  const year = now.getFullYear()
  const month = String(now.getMonth() + 1).padStart(2, '0')
  const day = String(now.getDate()).padStart(2, '0')
  const hours = String(now.getHours()).padStart(2, '0')
  const minutes = String(now.getMinutes()).padStart(2, '0')
  const seconds = String(now.getSeconds()).padStart(2, '0')
  
  // 格式：YYYY/MM/DD HH:MM:SS
  formData.eventTime = `${year}/${month}/${day} ${hours}:${minutes}:${seconds}`
}
```

**功能特点**：
- 自动计算当前时间减去2天
- 保持与"插入当前时间"相同的格式（YYYY/MM/DD HH:MM:SS）
- 自动处理月份和年份的边界情况（如跨月、跨年）

## 使用场景

### 典型用例

1. **正常数据录入**
   - 点击时钟图标按钮 → 插入当前时间

2. **反追单数据处理**
   - 点击"-2天"按钮 → 自动填充两天前的时间
   - 适用于需要模拟历史数据的场景

### 示例

假设当前时间是：2026/05/14 15:30:45

- 点击"插入当前时间"：`2026/05/14 15:30:45`
- 点击"-2天"：`2026/05/12 15:30:45`
