# OPMPlatform 功能说明文档

## 简介
OPMPlatform 是一款文档格式处理与转换工具，主要用于功能需求文档的标准化处理，支持 Markdown 与 Word 之间的格式转换、文档格式化等功能，帮助用户快速生成符合规范的文档。


## 环境准备
- 安装 Python 3.8 及以上版本
- 执行依赖安装命令：
  ```bash
  pip install flask python-docx bootstrap-flask beautifulsoup4 markdown
  ```


## 功能说明

### 1. 文档格式化工具
将普通 Word 文档按照标准模板格式进行规范化处理，支持：
- 序号自动格式化
- 文字加粗规则统一
- 字号大小标准化
- 段落换行优化
- 多级标题结构调整（Heading 1-5）

**使用流程**：
1. 上传标准 Demo 模板（需包含 Heading 1-5 层级结构）
2. 点击「开始格式化文档」
3. 上传需要格式化的 Word 文档
4. 下载格式化后的文档


### 2. Markdown 与 Word 互转
#### Markdown 转 Word
- 上传 Markdown 文件（.md）
- 系统自动转换为符合规范的 Word 文档（.docx）
- 保留原文档的标题结构、列表、代码块和表格等元素

#### Word 转 Markdown
- 上传 Word 文档（.docx）
- 转换为 Markdown 格式文本
- 保留原文档的标题层级、加粗、斜体等格式


### 3. 功能需求文档转换
将普通文档转换为标准 Demo 格式，遵循「一级分类→二级分类→三级分类→功能点名称→功能点计数项」的层级结构，自动提取并格式化：
- 场景说明
- 功能描述
- 系统界面
- 输入/输出信息
- 处理过程
- 涉及数据文件（FTR/RET）


## 使用流程

### 步骤 1：准备 Demo 模板
- 在 `static/templates/` 目录下创建空白 Demo 模板（按 Heading 1-5 设置层级）
- 或通过系统页面下载默认模板并修改
- 上传自定义模板至系统

### 步骤 2：上传待转换文档
- 选择对应功能模块（格式化/Markdown转换/需求转换）
- 上传包含功能需求的 Word 文档或 Markdown 文件
- 系统自动解析文档结构

### 步骤 3：获取转换结果
- 转换完成后，点击下载按钮获取处理后的文档
- 生成的文档将严格遵循模板格式规范


## 注意事项
- 支持的文件格式：Word（.docx）、Markdown（.md）
- 文档内容需包含明确的标题结构和关键字段（如"功能描述"、"一级分类"等）
- 转换前建议使用「格式检查」功能确认文档结构匹配度
- 纯图片或扫描件文档无法进行结构化转换
