# 部署步骤

## 环境准备
- 安装 Python 3.8+
- 执行 `pip install flask python-docx bootstrap-flask`

## 预生成 Demo 模板
在 `static/templates/` 目录下创建空白 Demo 模板（按 Heading 1-5 设置层级）

## 启动服务
运行 `python app.py`，访问 http://localhost:5000 进入系统

# 使用流程

## 步骤 1：导入 Demo 模板
- 访问首页，点击「步骤 1」的「上传 Demo 模板」
- （可选）先下载「标准 Demo 模板」，按模板填写示例数据
- 上传 Demo 文件，系统验证格式（通过后提示成功）

## 步骤 2：上传待转换文档
- 点击「步骤 2」的「上传待转换文档」
- 上传包含功能需求的普通 Word 文档（需包含 "一级分类""功能描述" 等关键词字段）
- 系统自动解析并转换为 Demo 格式

## 步骤 3：下载转换结果
- 转换成功后，点击「下载转换后的 Demo 格式文档」
- 下载的 Word 文件将严格遵循「一级分类→二级分类→三级分类→功能点名称→功能点计数项」格式
