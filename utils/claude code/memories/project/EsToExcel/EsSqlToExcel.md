# 使用场景
### 查询sql
```
POST /_sql?format=txt
{
  "query": """
    SELECT 
      "CREATION_EVENT_TIME",
      "EVENT_FP",
      "ORDER_ID",
      "DISPATCH_INFO.DISPATCH_REASON"
    FROM "mw_em_master"
    WHERE 
      "CREATION_EVENT_TIME" >= '2026-04-16 00:00:00'
      AND "ORDER_ID" IS NOT NULL
      -- 使用keyword子字段做精准匹配（派单状态）
      AND "DISPATCH_INFO.JT_DISPATCH_STATUS_DESC.keyword" = '自动派单成功'
      AND "DISPATCH_INFO.DISPATCH_REASON" not in ('通知单转处理单成功')
      AND "NETWORK_TYPE_ID" in ('1','4','5','12')
      AND "TODO_LIST_ID" IS NULL
      AND "ORDER_ID" LIKE 'FJ-076%'
    ORDER BY "CREATION_EVENT_TIME" DESC
  """
}
```
