# Word 转 Excel - 快速启动指南

## 🚀 5 分钟快速上手

### 1️⃣ 启动应用

```bash
cd /Users/linziwang/PycharmProjects/wordToWord
./start_prod.sh
```

应用将在 http://127.0.0.1:5001 启动

### 2️⃣ 访问功能页面

在浏览器中打开：
```
http://127.0.0.1:5001/word-to-excel/
```

### 3️⃣ 上传文档

1. 点击 **"选择文件"** 按钮
2. 选择要转换的 Word 技术规范书 (.docx 格式)
3. 点击 **"上传并转换"** 按钮
4. 等待转换完成（通常 5-15 秒）
5. 点击 **"下载 Excel 文件"** 保存结果

## 🎯 文档格式要求

### ✅ 正确的编号格式

```
3.1.1.2    监控管理应用          ← 一级分类
3.1.1.2.1  分级调度管理          ← 二级分类
3.1.1.2.1.1  分级调度派发        ← 功能模块
3.1.1.2.1.1.1  派发智能体自动修复  ← 功能点名称
```

### ❌ 错误的编号格式

```
一、监控管理应用
(1) 分级调度管理
第 3.1.1.2 节 功能说明
```

## 💡 常见问题

### Q: 上传后没有提取到任何内容？

**A**: 检查文档是否包含类似 `3.1.1.2.1` 的层级编号。

### Q: 支持 .doc 格式吗？

**A**: 不支持。请使用 Word 另存为 `.docx` 格式。

### Q: 文件最大支持多大？

**A**: 最大 50MB。一般技术规范书都在此范围内。

### Q: 转换需要多长时间？

**A**: 
- 小文件 (< 1MB): < 5 秒
- 中等文件 (1-10MB): 5-15 秒
- 大文件 (> 10MB): 15-30 秒

## 🔧 命令行使用（可选）

### Python 脚本转换

```python
from utils.word_to_excel import quick_convert

# 简单转换
excel_path = quick_convert('技术规范书.docx')
print(f'转换成功：{excel_path}')
```

### 指定输出目录

```python
excel_path = quick_convert(
    '技术规范书.docx',
    output_dir='./output'
)
```
