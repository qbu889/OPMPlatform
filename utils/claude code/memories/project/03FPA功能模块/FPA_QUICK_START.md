# FPA预估表生成器 - 快速开始指南

## 🚀 5 分钟快速上手

### 第一步：准备需求文档

将你的需求规格说明书保存为 Markdown(.md) 格式。例如:

```markdown
## 监控管理应用

### 故障监控应用

#### 集团事件对接

##### 网络事件业务影响

###### 家宽业务 OLT 脱管场景业务采集

**功能描述：**进行家宽业务 OLT 脱管场景业务采集

**系统界面：**无

**输入：**系统自动实时触发

**输出：**查询出满足条件的事件数据

**处理过程：**系统自动实时触发，进行家宽业务 OLT 脱管场景业务采集。

**本事务功能预计涉及到 1 个内部逻辑文件，0 个外部逻辑文件**

本期新增/变更的内部逻辑文件：家宽业务影响指标清单表

本期涉及原有但没修改的内部逻辑文件：网络事件表

本期新增/变更的外部逻辑文件：无

本期涉及原有但没修改的外部逻辑文件：无
```

### 第二步：启动系统

确保 Flask 应用正在运行:

```bash
cd /Users/linziwang/PycharmProjects/wordToWord
python app.py
```

访问：http://localhost:5001

### 第三步：上传并生成

1. 在首页点击 **"FPA预估表生成"** 卡片
2. 拖拽你的.md 文件到上传框
3. 点击 **"开始生成 FPA预估表"**
4. 等待解析完成 (通常几秒钟)
5. 点击下载生成的 Excel 文件

### 第四步：查看结果

打开下载的 Excel 文件，你将看到:

- **工作表**: FPA功能点估算
- **列**: 一级分类、二级分类、三级分类、功能点名称、功能点计数项等 17 列
- **汇总**: 底部显示功能点总数和文件统计

## 🔧 故障排除

### 问题 1: 解析结果为空

**症状**: 提示"未能从文档中提取到功能点信息"

**解决方法**:
1. 检查文档是否包含 `######` 级别标题
2. 确认功能描述字段格式正确
3. 确保文档编码为 UTF-8

### 问题 2: 文件统计不准确

**症状**: 内部/外部逻辑文件数量不对

**解决方法**:
1. 检查文件格式是否正确:
   ```
   **本事务功能预计涉及到 X 个内部逻辑文件，Y 个外部逻辑文件**
   ```
2. 多个文件之间用逗号或分号分隔
3. 避免使用特殊符号

### 问题 3: Excel 格式混乱

**症状**: 下载的 Excel 表格格式不正确

**解决方法**:
1. 确保安装了 openpyxl 库：`pip install openpyxl`
2. 检查输出目录是否有写权限
3. 尝试重新生成

### 常见问题解答

**Q: 为什么有些功能点解析不出来？**
A: 检查标题级别是否正确，应该是 5-6 级标题 (##### 和 ######)

**Q: 可以处理 Word文档吗？**
A: 当前版本仅支持 Markdown 格式，需要先将 Word 转为.md

**Q: 如何自定义 Excel 模板？**
A: 修改 `generate_fpa_excel()` 函数中的样式设置
