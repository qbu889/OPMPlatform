## 🚀 使用流程

### 首次使用

```bash
# 1. 初始化数据库
python scripts/init_deploy_config.py

# 2. 启动应用
python app.py

# 3. 访问配置管理页面
http://localhost:5173/deploy-config
```

### 修改配置

1. 登录系统（需要管理员权限）
2. 访问配置管理页面
3. 选择对应的分类标签页
4. 修改配置值
5. 点击"保存"或"保存所有修改"
6. （可选）点击"测试"验证SSH连接

### 在代码中使用配置

```python
from utils.deploy_helper import DeployHelper

# 获取远程服务器配置
config = DeployHelper.get_remote_config()
print(config['remote_host'])  # 从数据库读取，无需硬编码

# 构建SSH命令
ssh_cmd = DeployHelper.build_ssh_command('ls -la')
# 输出: ssh -p 22 root@8.146.228.47 'ls -la'
```

## 📝 技术要点

### 1. 配置缓存优化

为避免频繁查询数据库，可以添加缓存：

```python
from flask_caching import Cache

cache = Cache(config={'CACHE_TYPE': 'simple'})

@cache.cached(timeout=300, key_prefix='deploy_config_{key}')
def get_cached_config(key):
    return DeployConfig.get_config(key)
```

### 2. 配置验证

添加配置值验证逻辑：

```python
def validate_config(key, value):
    if key == 'ssh_port' and not (1 <= int(value) <= 65535):
        raise ValueError('SSH端口必须在1-65535之间')
    if key == 'remote_host' and not is_valid_ip(value):
        raise ValueError('无效的IP地址')
```

### 3. 配置历史

记录配置变更历史：

```sql
CREATE TABLE deploy_config_history (
    id INT AUTO_INCREMENT PRIMARY KEY,
    config_key VARCHAR(100),
    old_value TEXT,
    new_value TEXT,
    changed_by INT,
    changed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
